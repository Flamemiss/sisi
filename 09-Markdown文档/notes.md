# Python学习笔记

## 第一章：基础语法

### 1.1 变量

Python中定义变量不需要声明类型：

```python
name = "张三"
age = 25
is_student = True
```

### 1.2 数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| int | 整数 | `10` |
| float | 浮点数 | `3.14` |
| str | 字符串 | `"hello"` |
| bool | 布尔值 | `True` |
| list | 列表 | `[1, 2, 3]` |

### 1.3 常用函数

- `print()` - 输出内容
- `input()` - 获取输入
- `len()` - 获取长度
- `type()` - 获取类型
